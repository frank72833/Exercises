# Bank
## Requirements
- Deposit and Withdraws
  - Transaction needs to go through teller (employee in physical location)
- Customer needs accounts
- Physical locations
  - HQ + Branches
  - Add new branches
- Security: Every day the bank takes X% to the HQ

## Classes
- Customer
  - Performs transaction
- Transaction
    - Deposit
    - Withdraw
    - Open Account
- Employee (teller)
- Account
  - Id
  - Name
  - Balance
- Bank (HQ)
  - Physical cash
- BankBranch (Branch)
  - Physical cash
